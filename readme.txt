Before you can use ARPGA, follow these steps:

1. Extract the zip file.
2. Open "ARPGA Installer"
3. If XAMPP is not installed, install XAMPP. 
4. To install, open the "xampp-windows-x64-8.2.12-0-VS16-installer.exe"
5. After the installatin, open XAMPP Control Panel. 
6. Start Apache, and MySQL.
7. In MySQL, click "admin". it will redirect to the database portal.
8. In the portal, click "New"
9. On the database name textbox, type "login_db"
10. Click "Create"
11. Click the database "login_db"
12. Click "Import"
13. On Import page, click "Choose File"
14. Navigate the extracted folder.
15. Select "login_db" file
16. Click "Import" at the bottom of the Import page.
17. You can now use ARPGA.
